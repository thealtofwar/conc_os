# conc_os

An OS where VMs are first-class citizens, i.e. a type-1 hypervisor that supports scale-to-zero and other operations. We can't do this yet.

## Features + Roadmap

- [x] Memory allocation
- [x] Async
- [ ] Networking
- [ ] Disk
- [ ] Launching single VM
- [ ] Launching multiple VMs
- [ ] Scaling and scheduling VMs
- [ ] Scale-to-zero
- [ ] Fast scale-to-zero
- [ ] Scale-to-zero for large number of VMs
